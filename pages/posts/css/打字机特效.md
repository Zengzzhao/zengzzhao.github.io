## css

```css
.typing {
    overflow: hidden;
    border-right: .15em solid #000000;
    white-space: nowrap;
    margin: 0 auto;
    letter-spacing: .15em;
    animation:
        typing 3s steps(30, end) infinite,
        blink-caret .3s step-end infinite;
}

@keyframes typing {
    0% {
        width: 0;
    }

    40% {
        width: 100%;
    }

    70% {
        width: 100%;
    }

    90% {
        width: 0;
    }

    100% {
        width: 0;
    }
}

@keyframes blink-caret {

    from,
    to {
        border-color: transparent
    }

    50% {
        border-color: #000000
    }
}
```
